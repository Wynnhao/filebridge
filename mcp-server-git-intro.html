mcp-server-git

A Model Context Protocol server for Git repository interaction and automation

MCP Server STDIO 可部署 MIT License Early Dev

📖 Overview

mcp-server-git 是一个基于 MCP（Model Context Protocol）协议的服务器，让大语言模型能直接读取、搜索、操作 Git 仓库。通过它，AI agent 可以像人一样查看仓库状态、提交代码、切换分支——无需手敲命令。

⚠️ 目前处于早期开发阶段，功能和可用工具可能随迭代持续变化。

🔧 Tools（12 个工具）

git_status
查看工作区当前状态
repo_path string
git_diff_unstaged
查看未暂存的改动
repo_path string
git_diff_staged
查看已暂存的改动
repo_path string
git_diff
对比分支或 commit 之间的差异
repo_path · target string
git_commit
提交变更到仓库
repo_path · message string
git_add
将文件加入暂存区
repo_path · files string[]
git_reset
取消所有已暂存的变更
repo_path string
git_log
查看 commit 日志
repo_path · max_count number (可选, 默认10)
git_create_branch
创建新分支
repo_path · branch_name · start_point (可选)
git_checkout
切换分支
repo_path · branch_name string
git_show
查看某个 commit 的详细内容
repo_path · revision string
git_init
初始化一个新 Git 仓库
repo_path string

⚡ Installation

Using uv（推荐）

# 无需单独安装，直接运行
uvx mcp-server-git

Using pip

pip install mcp-server-git
# 安装后运行：
python -m mcp_server_git

Docker

docker build -t mcp/git .
docker run --rm -i \
  --mount type=bind,src=/path/to/repo,dst=/workspace \
  mcp/git

⚙️ Configuration

以下为 Claude Desktop / VS Code 配置示例：

{
"mcpServers": {
"git": {
"command": "uvx",
"args": ["mcp-server-git"]
}
}
}
